# Attested English Coda Clusters

Sources: English phonotactic literature, Hammond (1999), Clements & Keyser (1983)

## 2-consonant codas

### Liquid/rhotic + obstruent
lp (help), lb (bulb), lt (melt), ld (held), lk (milk), lg (only in borrowed),
lf (self), lv (solve), ls (else), lz (bells*), lθ (health),
lʃ (Welsh), ltʃ (belch), ldʒ (bulge)
rp (warp), rb (orb), rt (art), rd (card), rk (park), rg (berg),
rf (scarf), rv (carve), rs (horse), rz (bars*), rθ (earth),
rʃ (harsh), rtʃ (arch), rdʒ (gorge)

### Liquid + nasal
lm (film), ln (kiln)
rm (arm), rn (barn), rl (Carl)

### Nasal + homorganic stop
mp (lamp), mb (lamb), nt (tent), nd (tend), ŋk (think), ŋg (ring... rare in coda cluster)

### Nasal + fricative
mf (nymph), ms (hams*), mz (hams*), mθ (warmth)
nf (? — rare), ns (dance), nz (lens), nθ (month)
ŋs (songs*), ŋz (songs*), ŋθ (length)

### Nasal + affricate
ntʃ (lunch), ndʒ (lunge)

### Obstruent + /s,z/ (includes morphological)
ps (lapse), bz (cabs*), ts (cats*), dz (ads*), ks (box),
fs (laughs*), vz (moves*), θs (myths*), ðz (bathes*),
ʃ? — not standard
tʃ+... — not as first element of multi-coda typically

### Stop + stop (limited, same-voicing)
pt (apt), kt (act)

### Fricative + stop (limited)
ft (left), sp (wasp), st (nest), sk (desk)

### Stop + θ
pθ (depth), kθ (? — rare)

### Fricative + θ
fθ (fifth)

## 3-consonant codas

### Liquid + obstruent + /s,z/
lps (helps*), lts (melts*), lds (fields*), lks (milks*),
lfs (elves... no, selfs?), lvz (solves*), lθs (healths*)
rps (warps*), rts (arts*), rds (cards*), rks (parks*),
rfs (scarfs*), rvz (carves*), rθs (earths*)

### Liquid + stop + stop/fricative
lpt (sculpt), lkt (? — rare)

### Liquid + fricative + stop/s
lfθ (twelfth), lst (? — rare)

### Nasal + stop + /s,z/
mps (bumps*), mpt (? — exempt?), nts (ants*), nds (bands*),
ŋks (thanks*), ŋkθ (length+θ? rare)

### Nasal + affricate + /s,z,t/  
ntʃt (lunched*), ndʒd (lunged*)

### Obstruent + /s/ + stop — rare
kst (next, text), kts (facts*)

### Fricative + stop + /s/
fts (lifts*), sts (fists*), sps (wasps*), sks (asks*), skt (asked*)

## 4-consonant codas (with appendant)
lpts (sculpts*), lfθs (twelfths*), ŋkθs (lengths*), ksts (texts*), mpts (? — v. rare)

* = morphological (plural/past/3sg) but phonotactically real
